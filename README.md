# Modern Login Form UI 🔐

ဒီ Project ဟာ **HTML5** နဲ့ **CSS3** ကိုသာ အသုံးပြုပြီး ဖန်တီးထားတဲ့ ခေတ်မီပြီး မျက်စိပသာဒဖြစ်စေမယ့် **Login Form** ဒီဇိုင်းဖြစ်ပါတယ်။ User Interface (UI) ပိုင်းကို သေသပ်လှပအောင် အဓိကထား တည်ဆောက်ထားပါတယ်။

---

## 🌟 Features (ထူးခြားချက်များ)

* **Clean & Professional UI:** ရိုးရှင်းပြီး ကြည့်ကောင်းတဲ့ အရောင်အသွေးနဲ့ အချိုးအစားကို အသုံးပြုထားခြင်း။
* **Icon Integration:** Input fields များတွင် သက်ဆိုင်ရာ Icon လေးများဖြင့် သေသပ်စွာ ဖော်ပြထားခြင်း။
* **Responsive Design:** မိုဘိုင်းလ်ဖုန်းနှင့် ကွန်ပျူတာ Screen အရွယ်အစားအားလုံးတွင် အဆင်ပြေစွာ ကြည့်ရှုနိုင်ခြင်း။
* **User Friendly:** အသုံးပြုသူများ နားလည်လွယ်သော Layout ပုံစံဖြင့် ဖန်တီးထားခြင်း။

## 🛠️ Technologies Used (အသုံးပြုထားသော နည်းပညာများ)

* **HTML5** - ဝဘ်စာမျက်နှာ၏ တည်ဆောက်ပုံ (Structure) အတွက်။
* **CSS3** - ဒီဇိုင်းပိုင်းဆိုင်ရာ (Styling, Layout & Animations) အတွက်။

## 🚀 How to Run (အသုံးပြုပုံ)

1.  Repository ကို **Clone** လုပ်ပါ သို့မဟုတ် **Download ZIP** ကို နှိပ်၍ ဒေါင်းလုဒ်ဆွဲပါ။
2.  ရရှိလာသော Folder ထဲရှိ `index.html` ဖိုင်ကို Browser ဖြင့် ဖွင့်လှစ်၍ ကြည့်ရှုနိုင်ပါသည်။
